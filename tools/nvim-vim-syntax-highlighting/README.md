Directions
---------- 
1. Place the `ftdetect` and `syntax` directories into your Neovim config
    - on Unix systems, this can be found at `~/.config/nvim`
    - if `~/.config/nvim/ftdetect` and `~/.config/nvim/syntax/` already exist, just copy the respective `bear.vim` files into those existing directories
2. Now, you will have automatic syntax highlighting for bear-lang source code (in `*.br` and `*.bear` files)!
